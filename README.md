# 📊 Data Science Salary Dashboard

An interactive Excel dashboard that analyses **30,000+ real-world job postings** 
to surface salary insights across job titles, countries, and employment types.

## 🔍 What It Does

- Select any **job title**, **country**, and **employment type** using dropdowns
- Dashboard instantly displays the **median salary** for that combination
- Clean, professional layout built for real decision-making

## 🛠️ Skills Demonstrated

| Skill | Usage |
|-------|-------|
| XLOOKUP | Dynamic salary lookup across datasets |
| Data Validation | Dropdown menus for user input |
| Multi-sheet Architecture | Separated raw data from dashboard layer |
| Data Cleaning | Handled missing values and inconsistencies |

## 📁 File

| File | Description |
|------|-------------|
| `Salary_Dashboard.xlsx` | Main Excel file — open and interact directly |

## 💡 How to Use

1. Download `Salary_Dashboard.xlsx`
2. Open in Microsoft Excel (2016 or later)
3. Use the dropdowns to select your filters
4. View the median salary result instantly

## 📌 Data Source

Dataset: [Data Science Jobs & Salaries](https://www.kaggle.com/) — Kaggle  
Records: 30,000+ job postings across multiple countries
